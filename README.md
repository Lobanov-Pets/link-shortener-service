Необходимо написать сервис для сокращения ссылок наподобие [tinyurl](https://tinyurl.com/app). Нужен только HTTP API. Основные задачи:

1. Написать приложение с использованием Spring Boot

2. Принимать на вход длинные ссылки и возвращать короткие

3. При обращении к сервису по короткой ссылке через браузер делать редирект на исходную ссылку

4. В качестве БД использовать PostgreSQL

5. Для БД моделей используйте Hibernate/JPA

6. Миграции БД нужно делать или через Liquibase или через Flyway

7. Настроить дев среду через docker/docker-compose.

8. Написать юнит и интеграционные тесты

9. Использовать maven или gradle

10. Проект должен быть выложен на github

Бонус:

1. Сделать alias для ссылок, чтобы можно было создавать читаемые ссылки.
2. Сделать ttl на ссылки. То есть возможность создавать ссылки с временем жизни или же вечные ссылки.
